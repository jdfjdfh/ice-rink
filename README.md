## Что нужно делать перед проверкой:

Скачать проект, распаковать и:
1. cd 'путь'
2. composer update
3. copy .env.example .env
3. php artisan key:generate
4. Потом заходим в .env и спускаемся в самый низ и меняем данные ЮКассы, которые были на сайте</br>
5. php artisan migrate --force --seed
6. php artisan serve
Тестовый юзер изначально там вводится уже почта, нужно пароль только вести он: "admin123".
После проверяем работу, но, нужно добавить коньки через админку (если будете проверять по пути, что нужно еще коньки), и все.
